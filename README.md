# Проектирование БД для учёта производственных процессов
## ООО «Центр Независимых Технологий Плюс»

### Описание проекта
Данный репозиторий содержит материалы по проектированию и реализации реляционной базы данных для автоматизации учёта производственных процессов промышленного предприятия.

### Цель проекта
Создать централизованную систему учёта, которая позволит:
- Отслеживать выполнение производственных заказов
- Контролировать использование оборудования и материалов
- Учитывать рабочее время сотрудников
- Формировать аналитические отчёты

### Структура репозитория
- `/docs` - документация и отчёты по дням практики
- `/sql` - SQL-скрипты для создания БД и работы с данными
- `/erd` - ER-диаграммы и схемы базы данных

### Ход работы
**День 1:** Анализ предметной области, выделение сущностей, создание концептуальной ER-диаграммы

### Используемые технологии
- Система управления базами данных: PostgreSQL/MySQL
- Средства проектирования: draw.io
- Контроль версий: Git/GitHub
